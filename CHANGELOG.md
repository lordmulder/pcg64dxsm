# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## Version 1.2.0

### Added

- Added new option `--hex` to output the generated bytes as Hex-encoded string.
- Added simple build scripts for the MacOS X platform.

## Version 1.1.0

### Added

- Added new option `--fast` to use the **Mcg128Xsl64** (`pcg64_fast`) generator.
- Added new option `--count` to limit the output size.
- Added integration tests and benchmarks.
- Added simple build scripts for the Windows and Linux platforms.
